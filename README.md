## Usage

### To install

```bash
yarn
```

### Run in development mode

```bash
yarn dev
```

### Run in Storybook mode

```bash
yarn storybook
```

## Features

- React + Typescript + Vite + Storybook
- Tailwind CSS
- MSW - Mock Service Worker API mocking
- Dark mode
- Tailwind CSS class sorting, merging and linting.
- 100% customizable
